# DreamHole

> Расширение Chrome для аналитики вакансий на hh.ru

![icon](extension/icons/icon128.png)

DreamHole показывает скрытую информацию прямо на страницах hh.ru: сколько висит вакансия, менялась ли зарплата, выше или ниже она рынка, и что говорят о найме в эту компанию.

---

## Возможности

**Аналитика вакансии**
- Возраст вакансии и история переподъёмов
- Сравнение зарплаты с рынком (p25 / медиана / p75 по роли, опыту, локации)
- История изменений зарплаты и требований
- Медианное время закрытия похожих вакансий

**Профиль компании**
- Количество активных вакансий прямо сейчас
- Тренд найма по месяцам
- Как часто переоткрывают одну и ту же позицию

**Отзывы о найме**
- Этапы, сложность, оценка HR, результат (оффер / отказ / гхост / в процессе)
- Реакции: 👍 👎 🔥 💩 🤡
- Анонимность через SHA-256 хэш — один отзыв на пользователя на компанию

## Установка (без Chrome Web Store)

1. Скачай репозиторий: **Code → Download ZIP**, распакуй
2. Открой `chrome://extensions`
3. Включи **Режим разработчика** (переключатель справа вверху)
4. Нажми **Загрузить распакованное расширение**
5. Выбери папку `extension/` внутри скачанного архива

Расширение подключается к публичному API — ничего дополнительно настраивать не нужно.

## Стек

| Слой | Технологии |
|---|---|
| Расширение | Vanilla JS, Chrome Manifest V3, Web Crypto API |
| Backend | Python, FastAPI, AsyncPG, Aiohttp |
| БД | PostgreSQL |
| Краулер | Asyncio, ежедневно, ~200k вакансий |

---

## Запуск backend локально

```bash
cd backend
cp .env.example .env
# заполни DATABASE_URL и ADMIN_TOKEN в .env

pip install -r requirements.txt
python main.py
```

PostgreSQL должен быть запущен локально. Структура БД создаётся автоматически при первом запуске.

---

## Деплой на сервер

```bash
cp .env.deploy.example .env.deploy
# заполни DEPLOY_HOST, DEPLOY_USER, DEPLOY_PASS

python deploy.py
```

Скрипт устанавливает зависимости, настраивает PostgreSQL и создаёт systemd-сервис.

## Структура проекта

```
extension/       # Chrome расширение (Manifest V3)
  popup.js       # Основная логика UI
  content.js     # Инжекция панели на страницы hh.ru
  manifest.json

backend/
  api.py         # FastAPI endpoints
  database.py    # Работа с PostgreSQL
  crawler.py     # Ежедневный краулер hh.ru
  main.py        # Точка входа (API + краулер параллельно)
  config.py      # Конфигурация

deploy.py        # SSH деплой на VPS
```
